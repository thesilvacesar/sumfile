### Get file hash using cli api

Get the hash of file:

*If tool it's as variable environment of windows*

``sumfile -f __FILE__``

*Else tool it's as variable environment of windows*

* Go to path of api
* Open the terminal on path and type:

``sumfile -f __FILE__``

### Or use standard mode

``sumfile``
